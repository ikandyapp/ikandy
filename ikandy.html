How sources work 

    
IKANDY can connect to four different sources. Each one offers a different feature set depending on what its underlying API exposes.


    
Full features

    
Spotify Premium — playlist browse, search, drill-in to tracks, all shuffle modes, transport controls, album art, lyrics. Requires a Premium subscription for transport control. Free accounts get track info only.

    
foobar2000 — playlist browse with search, drill-in to tracks, every shuffle mode foobar supports (random, shuffle by track/album/folder, repeat playlist, repeat track), transport, album art, lyrics. Requires the foo_beefweb component.


    
Standard

    
VLC — transport, shuffle on/off, album art, lyrics. No library browse. Requires VLC’s Web interface to be enabled with a password.


    
Basic

    
Now Playing — reads whatever app is currently playing music on Windows (Spotify desktop, YouTube Music, Apple Music, Chrome/Edge tabs, etc.) via the system Media Controls. Transport works, but exact controls depend on what each source app supports — greyed-out buttons mean that app didn’t expose that command. No browsing, no shuffle, no album art. Windows only.


    
What about volume?

    
Volume control isn’t exposed by IKANDY — use the source app or your system mixer.


    
Which one should I use?

    
If you have Spotify Premium or use foobar2000, pick those for the richest experience. If you mostly play music in a browser tab or another app, use Now Playing. VLC is best if you’re playing local files in VLC specifically.
